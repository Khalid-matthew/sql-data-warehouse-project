# sql-data-warehouse-project
building a modern data warehouse with SQL Server, including ETL, data modeling, and analytics.
---

Welcome! This project demonstrates an end-to-end **Data Warehouse and analytics project** solution built with **SQL Server** and a **Power BI dashboard**.  
The goal is to collect data from operational systems (ERP/CRM), transform it into analytics-ready models, and deliver clear business insights through interactive reports.

---

## 🎯 Project Objectives
- Build a structured **Data Warehouse** for reporting and analytics
- Integrate data from **ERP** and **CRM** sources into a single source of truth
- Apply data cleaning, standardization, and business rules (ETL/ELT)
- Create a **Power BI dashboard** to track KPIs and trends (sales by time, product, country, branch, etc.)
- ---

## ✅ Project Requirements

### Tools & Technologies
- **SQL Server** (Database + Data Warehouse layer)
- **Power BI** (Data model + DAX measures + Dashboard)
- *(Optional)* SSMS / SSIS / SQL Agent for management and scheduling

### Skills Used
- Data modeling (Star schema: Facts & Dimensions)
- SQL (DDL/DML, views, stored procedures)
- ETL/ELT (staging → transformations → warehouse)
- Power BI (Power Query, relationships, DAX, visuals)
